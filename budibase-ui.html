My Role & The Business Context:

As an in-house Product Designer at Budibase, I designed a wide range of user interfaces for new and existing features. This project is a curated showcase of that work, focusing on my UI and interaction design skills.

Project Overview:

This page features a collection of short videos that highlight my user interface design work at Budibase. It's designed to demonstrate my ability to create modern, intuitive, and user-friendly interfaces for complex applications. Each video demonstrates a specific feature and the UI I designed for it.

Automation Branches

This video demonstrates the UI for creating and managing conditional branches in Budibase automations. The design focuses on clearly visualizing the logical flow, making it easy for users to build and debug complex workflows.

Permissions

This video showcases the permissions UI, which allows administrators to control access to different parts of their applications with precision. The goal was to create a clear and intuitive way to manage complex role-based access levels.

View Calculations

This video highlights the UI for creating view calculations. This powerful feature allows users to perform real-time calculations on their data. The interface was designed to be formula-friendly, feeling familiar to users of spreadsheet software.

Accounts B2

This video shows the "Accounts B2" interface. Please add a description for this feature.

JavaScript Generation

This video features the JavaScript generation UI. This allows users to write custom JavaScript directly within the Budibase editor, complete with syntax highlighting and an intuitive code-editing experience.

Generate with AI

This video showcases the "Generate with AI" feature, which uses AI to help users build applications faster. I designed the interface to be simple and conversational, allowing users to generate complex app components with natural language prompts.
